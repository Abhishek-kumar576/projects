# Diabetes Prediction Using Machine Learning & Deep Learning

## Overview
A predictive model that classifies whether a patient is likely to have diabetes based on medical and lifestyle features. The project compares traditional Machine Learning algorithms with Deep Learning models to evaluate accuracy and performance.

## Dataset
- *Source*: PIMA Indians Diabetes Dataset from UCI / Kaggle
- *Features*: 8 input variables including Glucose, BMI, Age, Blood Pressure, Insulin, Pregnancies, Skin Thickness, Diabetes Pedigree Function
- *Target*: Outcome - 0 = Non-diabetic, 1 = Diabetic
- *Records*: 768 samples

## Key Features
- *Data Preprocessing*: Handling missing values, feature scaling with StandardScaler, train-test split
- *Machine Learning Models*: Logistic Regression, Random Forest, Support Vector Machine, XGBoost
- *Deep Learning Model*: Feedforward Neural Network built with TensorFlow/Keras
- *Evaluation*: Accuracy, Precision
